Sample behavioral waveforms for design file "CL_fifo_16.vhd"

The following waveforms show the behavior of dcfifo megafunction for the chosen set of parameters in design "CL_fifo_16.vhd". The design "CL_fifo_16.vhd" has a depth of 32 words of 16 bits each. The fifo is in legacy synchronous mode. The data becomes available after 'rdreq' is asserted; 'rdreq' acts as a read request.
